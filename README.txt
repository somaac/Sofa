# Hospitality Team 48 - Sofa Lookup System
# فريق الضيافة رقم 48 - نظام البحث عن الصوفا

A mobile-friendly web application for looking up sofa numbers and determining their corresponding camp, room, color, and gender designation.

## Features ✨

- **22 Camps** with complete room ranges
- **5 Color Categories**: Blue, Green, Yellow, White, Red
- **Gender Detection**: Male (≤12), Female (>12)
- **Mobile-Responsive Design**
- **Arabic/English Bilingual Interface**
- **Real-time Search & Validation**

## Structure 📁

## Setup Instructions 🚀

1. **Clone or Download** this repository
2. **Create an `images` folder** in the root directory
3. **Add your logo images**:
   - Save the Labbeh logo as `images/labbeh-logo.png`
   - Save the Nusuk logo as `images/nusuk-logo.png`
4. **Open `index.html`** in your web browser

## GitHub Pages Deployment 🌐

1. **Upload all files** to your GitHub repository
2. **Go to Settings** → Pages
3. **Select Source**: Deploy from a branch
4. **Choose Branch**: main (or master)
5. **Your site will be available at**: `https://username.github.io/repository-name`

## Color Scheme 🎨

- **Primary Green**: #2F5233 (Labbeh brand color)
- **Gold/Yellow**: #A68B5B (Nusuk brand color)
- **Background**: Gradient from green to gold

## Camp Color Classifications

- **Blue**: Camps 1, 3, 4, 13, 15, 16
- **Green**: Camps 2, 5, 8, 11, 12, 22
- **Yellow**: Camps 9, 10, 18, 19, 21
- **White**: Camps 6, 7
- **Red**: Camps 14, 17, 20

## Usage 📱

1. Enter a 5-digit sofa number
2. Click "البحث - Search" or press Enter
3. View results: Camp, Room, Color, Gender
4. Clear error message for invalid numbers
